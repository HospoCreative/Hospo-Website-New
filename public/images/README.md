# Image Folders

Use these folders to update the website images without changing code.

Accepted file types:

- `.jpg`
- `.jpeg`
- `.png`
- `.webp`
- `.avif`

Main folders:

- `hero/background` - one large background image for the hero.
- `hero/collage` - four images for the hero collage.
- `gallery` - photo gallery slider.
- `reels` - fallback thumbnails for reels without embeds.
- `about` - one portrait/team image.
- `strip` - moving image strip near the bottom.

After adding or replacing images, refresh `http://localhost:3000`.
