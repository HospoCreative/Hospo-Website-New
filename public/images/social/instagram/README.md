# Instagram Feed Preview

Drop images here to update the Instagram preview grid.

The site shows the first six supported images alphabetically.

Best format:

- Square images for a classic Instagram grid
- High-quality feed images or post cover images
- Rename strongest/newest posts as `01.jpg`, `02.jpg`, `03.jpg`, etc.
